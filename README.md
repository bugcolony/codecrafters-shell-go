# 🐚 Shell in Go

A small Unix-like shell written in Go for the
[CodeCrafters **"Build Your Own Shell"** challenge](https://app.codecrafters.io/courses/shell/overview).

## ✨ Features

- 🧠 **Command parsing** for quoted args, escapes, redirects, and pipelines
- ⚙️ **Builtins** like `cd`, `pwd`, `echo`, `type`, `history`, and `jobs`
- 🚀 **External command execution** through your system `PATH`
- 🔀 **Pipelines** for chaining commands together
- 📄 **Redirects** for stdout and stderr
- 🌙 **Background processes** with job tracking
- ⌨️ **Tab completion** for commands and files
- 📦 **Shell variables** with `declare`

## 🎯 Challenge

This project follows the CodeCrafters shell track, where each stage builds up
another piece of a working POSIX-style shell.
